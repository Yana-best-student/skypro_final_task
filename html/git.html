Git

Управление файлами и папками

  • git -v - Эта команда выведет версию Git, если он правильно установлен.
  • git init - Инициализация (initialisation) репозитория — создание репозитория.
  • dir /ah - просмотр скрытого файла git. В него будут сохраняться все изменения в репозитории.
  • git status - команда для просмотра изменений в удаленном репозитьрии.
  • git clone ссылка с GitHub - Склонировать удаленный репозиторий с GitHub себе на компьютер.
  • git add "имя файла" - добавление файла в отслеживаемые.
  • git commit -m "Название коммита" операция записи изменений в репозитории Git.
  • git push - внесение изменений на удаленный репозиторий.
  • git log - вывести список всех наших коммитов.
  • git pull - поддтягивает изменения из удаленного репозитория в локальный.
